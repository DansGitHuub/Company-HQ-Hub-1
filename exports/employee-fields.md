# Employee Record Fields

## Identity
- Employee Number
- First Name
- Last Name
- Preferred Name
- Pronouns
- Date of Birth
- Profile Photo

## Contact
- Personal Email
- Personal Phone
- Address
- City
- State
- Zip

## Emergency Contact 1
- Name
- Relationship
- Phone

## Emergency Contact 2
- Name
- Relationship
- Phone

## Job / HR
- Job Title
- Department
- Employment Type (Full-time, Part-time, etc.)
- Start Date
- End Date
- Supervisor
- Work Location
- Status (Active, Inactive, etc.)

## Pay
- Pay Rate
- Pay Type
- Pay Period
- Payment Method
- Bank Name (last 4)
- Account Number (last 4)
- Routing Number (last 4)
- Account Type

## System Links
- User ID (linked login account)
- Candidate ID (linked hiring record)
