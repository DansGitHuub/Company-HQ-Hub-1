The app uses Replit SIDECAR object storage. Pattern (same as SOP media in routes.ts):
```ts
const SIDECAR = "http://127.0.0.1:1106";
const privateDir = process.env.PRIVATE_OBJECT_DIR || "";
// Get bucket:
const bucketRes = await fetch(`${SIDECAR}/object-storage/default-bucket`);
const { bucket_name: bucketName } = await bucketRes.json();
// Sign URL:
const signRes = await fetch(`${SIDECAR}/object-storage/signed-object-url`, {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ bucket_name: bucketName, object_name: objectPath, method: "PUT", expires_at: new Date(Date.now() + 900000).toISOString() })
});
const { signed_url } = await signRes.json();
// Upload:
await fetch(signed_url, { method: "PUT", headers: { "Content-Type": "image/jpeg" }, body: req.file.buffer });
// Public URL:
const publicUrl = `/objects/${bucketName}/${objectPath}`;
```

**Add these 3 endpoints inside registerCatalogRoutes:**

### POST /api/catalog/:id/image
upload.single("image"), requireAuth middleware
- objectPath = `catalog-images/${req.params.id}/primary-${Date.now()}.jpg`
- Upload to SIDECAR as above
- UPDATE catalog_items SET image_url = $1, updated_at = NOW() WHERE id = $2 RETURNING *
- Return the updated row

### POST /api/catalog/:id/option-image
upload.single("image"), requireAuth middleware
- req.body.option = the option name (e.g. "Red")
- objectPath = `catalog-images/${req.params.id}/option-${req.body.option.replace(/[^a-z0-9]/gi,'_')}-${Date.now()}.jpg`
- Upload to SIDECAR
- UPDATE catalog_items SET option_images = COALESCE(option_images, '{}')::jsonb || $1::jsonb, updated_at = NOW() WHERE id = $2 RETURNING *
  where $1 = JSON.stringify({ [req.body.option]: publicUrl })
- Return updated row

### DELETE /api/catalog/:id/option-image/:optionName
requireAuth middleware
- UPDATE catalog_items SET option_images = (option_images - $1), updated_at = NOW() WHERE id = $2 RETURNING *
  where $1 = decodeURIComponent(req.params.optionName)
- Return updated row

Also add GET /api/catalog/:id — return single item by id:
```sql
SELECT ci.*, COALESCE(json_agg(json_build_object('id', ct.id, 'name', ct.name)) FILTER (WHERE ct.id IS NOT NULL), '[]') AS tags
FROM catalog_items ci
LEFT JOIN catalog_item_tags cit ON cit.item_id = ci.id
LEFT JOIN catalog_tags ct ON ct.id = cit.tag_id
WHERE ci.id = $1
GROUP BY ci.id
```
