Build Materials Catalog Manager Web App

Goal
Create a web app to manage a landscaping materials catalog using categories, category specific fields, and a Smart Add Material workflow that only shows relevant inputs based on category.

1 Categories
Create these default categories and always sort alphabetically in all dropdowns

Aggregates & Gravel
Chemicals & Fertilizer
Hardscape & Pavers
Landscape
Mulch & Soil
Other
Perennials & Annuals
Trees & Shrubs

Category Rules
Categories must be editable without code changes
Admin can add new categories
Admin can rename categories and update all linked materials
Admin can delete categories with confirmation
If deleting a category with materials require user to either move materials to another category or delete them with an extra confirmation
No manual reordering categories remain alphabetical automatically

2 Category Management
Admin can
Add category
Rename category
Delete category with confirm checkbox
Bulk move materials from one category to another

3 Category Specific Fields
Admin can define fields per category with the following properties

Field name
Field type text number dropdown multiselect boolean date textarea image file url
Required yes or no
Default value optional
Help text optional
Options for dropdown or multiselect
Show in public catalog yes or no
Sort order within the category form

Admin can
Add a field to one category multiple selected categories or all categories
Edit a field at any time
Delete a field with confirmation
Option to hide field but keep existing data or permanently delete field and data

Fields must be editable later without breaking existing materials

4 Material Data
Each material supports

Name required
Category required
Status Active or Inactive
Primary image optional
Gallery images optional
Description optional
Vendor or supplier optional
Unit of measure optional
Tags optional
Custom field values based on category fields

No SKU item number or pricing fields exist anywhere in the system

5 Search and Filtering
Admin material list supports
Search by name vendor tags
Filter by category and status
Sort alphabetically by name
