REPLIT AGENT PROMPT — QUIZ ADAPTIVE DIFFICULTY
────────────────────────────────────────────────────────
Fix the Quizzes module to support adaptive difficulty. Currently all users get the same questions regardless of skill level. Here is the complete fix:
DATABASE CHANGES: Add these columns to the quiz_questions table: difficulty_level (integer 1–5, required), audience_roles (JSON array of roles this question applies to, e.g. ['crew_member','crew_lead']). Add these columns to quiz_attempts table: questions_served (JSON array of question IDs in order served), current_difficulty (integer, tracks where the user currently is in the adaptive sequence), highest_level_passed (integer), final_score_label ('foundational'|'competent'|'proficient'|'advanced'|'expert'). Run migrations.
ADAPTIVE ENGINE: Build a server-side function selectNextQuestion(quizId, userId, currentDifficulty, answeredQuestionIds, userRole). It: (1) Filters questions by quizId AND audience_roles includes userRole AND id not in answeredQuestionIds. (2) If last answer was correct, selects from difficulty = currentDifficulty + 1 (max 5). (3) If last answer was wrong, selects from difficulty = currentDifficulty. (4) If no questions remain at that difficulty, selects the nearest available difficulty. (5) Returns the selected question. Starting difficulty: New Hire = 1, Crew Member = 1, Crew Lead = 2, Manager = 3.
QUESTION CREATION: In the quiz question editor, add: difficulty level selector (1–5 with labels: Foundational/Recall/Application/Analysis/Expert) and audience roles multi-select. When the SOP Agent creates quiz questions automatically from an SOP, it must tag each question with difficulty and audience. Prompt GPT to generate at least 2 questions per difficulty level per quiz.
QUIZ EXPERIENCE: Update the quiz-taking UI: show the current difficulty level badge on each question (so the user knows it's getting harder). Show progress as 'Level X' not just a question counter. After a wrong answer, show the correct answer with an explanation before moving on. At the end, show: highest level reached, a label (Foundational/Competent/Proficient/Advanced/Expert), and which areas to review if they struggled. Save the highest_level_passed to the employee's training record.
MANAGER VIEW: In the quiz results section of HR/Manager view, show each employee's highest level passed per quiz. Flag anyone who hasn't passed Level 2 on safety-critical SOPs. Allow managers to require a minimum level to pass (e.g., Crew Leads must reach Level 4 on equipment SOPs).
